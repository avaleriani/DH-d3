#Clase 1 - Grafico de barras



###Pre-requisitios

_Node: < 8.9.3_

_Npm: < 5.5.1_

###Instalacion

`npm run pre`

`yarn install`

###Desarrollo frontend

`yarn run dev`

###Desarrollo server

`yarn run dev-server`

###Build de produccion y correr server

`yarn start`
